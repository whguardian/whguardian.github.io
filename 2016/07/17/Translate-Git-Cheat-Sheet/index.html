配置工具

为所有本地项目库设置用户信息

$ git config --global user.name "[name]"

配置个人的用户名

$ git congfig --global user,name "[email address]"

配置个人的邮箱

$ git config --color.ui auto

开启颜色提示帮助

创建项目库

新建一个项目库或从远程库克隆项目

$ git init [project-name]

创建一个新的本地项目库

$ git clone [url]
